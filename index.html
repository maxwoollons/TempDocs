Setting up the bot

The TempVoiceChannel Bot is one of the simplest bots to setup on discord.
1. Start off by ensuring the bot has been added to your discord server with the appropiate permissions required.
2. Once added type the command '!setup' to start the setup procedure for the bot to function correctly. '!setup' will create a new category and voice channel as displayed below.

3. Place the category in your desired place in the discord server to match the servers appearance.

Commands

Description

!setup

Initializes the setup procedure for the bot's main function.

!invite

The Bot will paste the invite link for bot

!lock

Locks the channel the message authors is currently in

!unlock

Unlocks the channel the message authors is currently in
